PolicyAPI Changes

In PolicyCenter 7.0, the PolicyRenewalAPI web service follows the WS-I standard instead of the earlier RPC-encoded standard for publishing web services. The API methods throw a SOAPException if a communication error occurs. For complete information, see “Policy Web Services” in the Integration Guide.
