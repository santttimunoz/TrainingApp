Changes to Team Tab Configuration

In PolicyCenter 8.0, configuration information about the Team tab has been moved from the Application Guide to the Configuration Guide.
The PolicyChangesStatisticsWindowSize parameter in config.xml was renamed to OtherWorkOrdersStatisticsWindowSize.
The User batch process is now the Team Screens batch process. This batch process collects summary counts for team screens.
