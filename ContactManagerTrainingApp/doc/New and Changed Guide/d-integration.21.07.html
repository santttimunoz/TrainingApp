New Populator Plugin

Use the PopulatorPlugin to populate entities from any String data format. For example, import XML data and populate a PolicyPeriod and its subobjects. There is a default plugin implementation that does this, but you can modify it to add line-specific information. Also you can modify the XML model that this default plugin uses, for example to add or remove properties from the XML.
IMPORTANT The PopulatorPlugin is deprecated as of version 8.0.0. See “Populator Plugin Deprecated and No Longer Called”.
