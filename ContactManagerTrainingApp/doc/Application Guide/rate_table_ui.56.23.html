Accessing Rate Routines

You must have the ratebookview permission to access rate routines.
1.
In PolicyCenter, select Administration → Rating → Rate Routines.
This screen has Policy Line, Code and Name fields.
The search returns all routines that contain the search criteria. The search is not case-sensitive. For example, if you enter premium in Name, the search returns all routines that contain that string, such as the PA Coverage Premium Algorithm routine.
To view all rate routines, select <not specified> for Policy Line and leave Code and Name blank.
To view rate routines that apply to all policy lines, select <applies to all> and leave Code and Name blank. In the search results, Policy Line set to <applies to all> appears blank.
2.
Enter your search criteria, then click Search.
For each search result, PolicyCenter displays the Name, Code, and Version.
