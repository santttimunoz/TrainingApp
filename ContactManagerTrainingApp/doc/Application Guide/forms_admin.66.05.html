Adding a Form Pattern

1.
Select Administration → Business Settings → Policy Form Patterns.
To view this menu item, you must have the View form pattern permission. The code for this permission is formpatview.
PolicyCenter displays the Policy Form Patterns screen.
2.
In the Search Results pane, click Add.
You must have the Create form pattern permission to view these buttons. The code for this permission is formpatcreate.
PolicyCenter displays the New Policy Form screen.
3.
Enter information about the form in the New Policy Form screen. See “Form Pattern or New Policy Form Screen” for details about this screen.
