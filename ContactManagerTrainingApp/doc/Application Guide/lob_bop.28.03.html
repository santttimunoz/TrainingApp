Offerings Screen for Businessowners

The Offerings screen contains a set of questions related to offerings.
Offerings let you define different product types for different types of buyers. Answers to the questions can affect which offerings are available. Offerings can filter parts of the product model such as policy terms, policy lines in a package policy, coverages, covterms, covterm options and packages, modifiers, and question sets.
In the default implementation, the answers to the questions determine the choices on the Offering Selection drop-down list.
For more information about offerings, see “Understanding Offerings”.
