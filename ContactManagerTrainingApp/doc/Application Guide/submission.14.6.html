Places to Create a Submission

PolicyCenter has several places in the user interface where you can create a submission. However, you must have an account before you create a submission. Generally, if you have not selected an account and want to create a submission, PolicyCenter guides you to select an account first. If you already have an account, you then confirm the producer, product, quote type, and number of desired policies. You can create a submission from:
•
Desktop → Actions
•
Account File → Actions
•
The Policy tab drop-down list
•
The Submission Manager screen
