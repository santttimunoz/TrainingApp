Copying a Submission

You can copy information from a submission to create a new submission. For additional information on this see “Copying Submission Information”.
1.
Find the submission you want to copy and navigate to the Submission Policy Info screen.
2.
From the Actions menu, select Copy.
Make any changes beginning with the Policy Info screen.
3.
After you finish, you can create a new version, save the draft, select from the bind options, or select from the close options.
