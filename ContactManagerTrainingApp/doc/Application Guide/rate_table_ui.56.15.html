Working with the Rate Table Editor

Use the Rate Table editor to view and edit the content of rate tables. You always edit rate tables in the context of a rate book. This requirement assures that the system can correctly manage the versions of rate tables and their relationships to rate books. See “Managing Rate Books and Rate Tables” for more information.
