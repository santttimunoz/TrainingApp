Accessing Summary Loss Information from the Claim System

In PolicyCenter, the underwriter can view loss history for accounts and policies. PolicyCenter retrieves the following loss summary information from the claim system:
 
•
Claim number
•
Value for remaining reserves
•
Loss date
•
Value for total paid
•
Loss cause
•
Value for total recoveries
•
Status
•
Value for net total incurred
•
Description
 
•
Information on litigation
 
•
Information on injuries
 
When integrated with ClaimCenter, the user can view the claim in ClaimCenter.
