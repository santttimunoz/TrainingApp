PolicyCenter Application Guide : Additional Features of PolicyCenter : Multicurrency Features

Multicurrency Features

Through multicurrency, PolicyCenter provides the ability to write policies that provide insurance for assets in different currencies. For example, a carrier offers a commercial property policy that can include properties in more than one country. With multicurrency, the policy values the properties in the currency of their locations.
