Types of Schedules

Although schedules can contain many types of information, PolicyCenter supports two broad types of schedules.
Schedules
Schedules of items can contain any number of columns. The scheduled items do not include coverage terms. None of the captured data is used in rating the policy.
In the base configuration, the General Liability line has examples of this type of generic schedule.

Schedules with Coverage Terms
Schedules of items with coverage terms can contain any number of columns. One or more of the columns capture coverage term data that is used to rate the policy. Examples of coverage terms include limit and deductible.
The base configuration does not provide any examples of schedules with coverage terms implemented using generic schedules. The Inland Marine line provides an example schedule with coverage terms, but the schedule is not implemented using a generic schedule. See the Scheduled Equipment schedule.
