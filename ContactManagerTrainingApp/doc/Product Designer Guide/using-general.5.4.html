PolicyCenter Product Designer Guide : Using Product Designer : Navigating in Product Designer : User Menu

User Menu

 
User Settings
Displays the User Settings page where you can change your user name, password, full name, user interface language, and regional formats.
My Changelists
Displays the My Changelists page where you can add and delete change lists, see which of your change lists is active, and reassign change lists to other users.
Log Out
Logs you out of Product Designer and displays the login screen.
