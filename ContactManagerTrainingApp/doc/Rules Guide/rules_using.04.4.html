Overview of PolicyCenter Rule Set Categories

A rule set category is the highest level of logical grouping of rules that are specific to a business function in ­PolicyCenter. A rule set category contains rule sets. In the base configuration, PolicyCenter provides the following rules set categories:
 
Rule set category
Description
Assignment
Manage the policy archive process. See “Assignment”.
Assignment
Determine the responsible party for an activity, for example. See “Assignment”.
Audit
Generate activities related to audits. See “Audit”.
Event Message
Handle communication with integrated external applications. See the PolicyCenter Integration Guide for information about events and event messaging. See “Event Message”.
Exception
Specify an action to take if an Activity or Job is overdue and is in the escalated state. See “Exception”.
Renewal
Govern behavior at decision points in the Renewal job flow, for example, accepting or rejecting a renewal request. See “Renewal”.
Validation
Check for missing information or invalid data on non-policy-related (platform) objects. See “Validation”.
