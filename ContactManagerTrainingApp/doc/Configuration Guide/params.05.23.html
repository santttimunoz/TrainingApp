Policy Exception Parameters

Guidewire provides the following configuration parameters in config.xml that limit how many days must pass before the Policy Exception Rules run a second time on any given PolicyPeriod entity.
See also
•
“Policy Exception Rules” in the Rules Guide

BoundPolicyThresholdDays

The number of days that must pass before the Policy Exception Rules run a second time on any given bound PolicyPeriod entity.
Default: 14

ClosedPolicyThresholdDays

The number of days that must pass before the Policy Exception Rules run a second time on any given closed PolicyPeriod entity.
Default: 14

OpenPolicyThresholdDays

The number of days that must pass before the Policy Exception Rules run a second time on any given open PolicyPeriod entity.
Default: 1
