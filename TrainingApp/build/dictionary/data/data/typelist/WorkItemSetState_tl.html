WorkItemSetState     (abtl_workitemsetstate)
State of a WorkItemSet
Attributes: Final

Referenced by:
WorkItemSet.State
Typecodes
Code Name Description Priority Internal Retired
Canceling Canceling a WorkItemSet goes into this state when the user requests that we cease further processing. Workers will continue processing current WorkItems, but will not start new ones.   true  
Completed Completed all WorkItems have been processed. Some WorkItems may have been successful and others may have failed   true  
CurrentlyPaused Paused the WorkItemSet is currently paused   true  
CurrentlyStarting Starting the WorkItemSet is currently starting up   true  
CurrentlyStuck Stuck the WorkItemSet appears to be stuck. Please cancel.   true  
Processing Processing the WorkItemSet is being worked on, i.e. there is at least one WorkItem that has not been completed yet   true
